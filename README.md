# GaBaus Sky-Logo Converter

High-performance web tool to convert images into Minecraft schematics using Obsidian and Crying Obsidian.

## Features
- **Zero UI Lag:** Uses Web Workers for chunked processing.
- **Extreme Scale:** Supports ultra high-resolution images.
- **Modern Format:** Outputs Sponge Schematic V2 (.schem).

## Getting Started

1. Install:
   ```bash
   npm install
   ```

2. Run:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000).

## Technical Details
- **Black (#000000):** Obsidian.
- **Purple (#9F81D6):** Crying Obsidian.
- **Transparency:** Air.
